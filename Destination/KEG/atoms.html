Buttons

If your call to action (cta) performs an action, such as submitting a form or triggering some javascript event, then use a <button> element. A Primary button should only be used economically as per one primary cta per page is best practice, default and complementary buttons can be used as secondary to a primary cta.

<button class="btn ld-btn-defaultCta" href="#">Default</button>
<button class="btn ld-btn-primaryCta" href="#">Primary</button>
<button class="btn ld-btn-compCta" href="#">Complementary</button>

Grids

The grid system is an aid, not a guarantee. It permits a number of possible uses for each designer. The grid system is based on Bootstaps 12 column grid system. All grids must be divisible by an even integer to maintain a 12 column grid structure. There are many ways to use a grid with various media breakpoints and helper classes. Headings

Headings or <h1>, <h2>, <h3> titles are used for the presentation and heirarchy of information on a page. Like a primary cta's it is best practice to keep your page titles limited to one h1 heading per page. It is common to use multiple headings, please denote the heirarchy of importance of each title to ensure best SEO practices.

h1. 30px Heading

h2. 24px Heading

h3. 18px Heading

h4. 16px Heading

Leading

The space between lines of type is traditionally known as leading. We define our web-leading as line-height, this controls the vertical rhythm of the typographic grid. Leading will be defined as part of the typography system with the exceptional helper class to break the rules where needed.
